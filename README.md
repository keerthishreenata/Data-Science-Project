Fuel Consumption Data Preprocessing using Python

Imagine you have a big notebook filled with information about thousands of cars, but some pages have missing numbers,
some words need to be changed into numbers, and everything is a little messy. Before a computer can learn from this data,
it must first be cleaned and organized. This project does exactly that. Using Python, Pandas, and Scikit-learn, 
I clean the vehicle fuel consumption dataset by finding missing values, filling them with suitable values, removing incomplete records, 
and converting text information like vehicle classes into numbers that computers can understand.
I also demonstrate different encoding techniques such as Mapping, Label Encoding, One-Hot Encoding, and Dummy Variables.
This project shows the complete data preprocessing workflow, which is one of the most important first steps in every Machine Learning and Data Science project.
